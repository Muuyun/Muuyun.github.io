机械设计基础-1

Posted on 2019-07-03 | In IE事务所 | | heat: ℃

机器及其组成

机械设计理论和方法

优化设计：寻求最佳设计方案 

有限元法：对强度、刚度、润滑、传热等进行数值计算 

可靠性设计：精确评价机械零件的强度和寿命 

机械CAD：应用计算机辅助解决机械设计的问题 

设计方法学：研究设计方法的学科

机器的分类：

原动机：实现能量转换（如内燃机、蒸汽机、电动机）
工作机：完成有用功（如机床、起重机等）

机器的基本组成：

  • 动力部分
  • 执行部分
  • 传动部分
  • 控制部分

机构

机构：具有确定相对运动的多件实物的组合体。

组成：

  1. 固定构件（机架）：支撑活动构件（运动构件）的构件；
  2. 原动件（主动件）：运动规律已知的活动构件；
  3. 从动件：机构中随原动件运动而运动的其它构件。

运动副及其分类

运动副： 1. 两个构件 2. 直接接触 3. 有相对运动

运动副的分类：

  1. 按引入的约束数分有： I级副，II级副，III级副，IV级副，V级副。

  1. 按相对运动范围分有：

    • 平面运动副 – 平面运动
    • 空间运动副 – 空间运动

  2. 按运动副元素分有：

    • 高副一点，线接触，应力高
    • 低副，面接触，应力低

平面机构的自由度

自由度 = 原动件数

机构自由度 = 3 - 约束数 = 自由构件的自由度数 - 约束数

平面机构自由度的计算

n – 机构中活动构件数

P{L} – 机构中的低副数

P{H} – 机构中的低副数

计算平面机构自由度的注意事项

  1. 复合铰链 – 两个以上的构件在同一处以转动副相连。

  2. 局部自由度： 构件局部运动所产生的自由度。

  3. 虚约束 – 对机构的运动实际不起作用的约束。

    • 两构件联接前后，联接点的轨迹重合

    • 两构件构成多个移动副，且导路平行

    • 两构件构成多个转动副，且同轴

    • 运动时，两构件上的两点距离始终不变

    • 对运动不起作用的对称部分

    • 两构件构成高副，两处接触，且法线重合

虚约束的作用：

  1. 改善构件的受力情况，如多个行星轮。
  2. 增加机构的刚度，如轴与轴承、机床导轨。
  3. 使机构运动顺利，避免运动不确定，如车轮。

习题：

  • 整体式连杆是一个构件，也是一个零件。
  • 一部机器可以只含有一个机构，也可以由数个机构组成。
  • 构件可以由一个零件组成，也可以由几个零件组成。
  • 机器与机构的本质区别是：是否能完成有用的机械功或转换机械能。
  • 起重吊钩属于专用零件。
  • 空气压缩机不属于机器的工作部分。

  • 机械中制造的单元称为零件，运动的单元称为构件，装配的单元称为机构。

  • 机构具有确定的相对运动条件是原动件数等于机构的自由度

  • 高副：1个约束 低副：2个约束
