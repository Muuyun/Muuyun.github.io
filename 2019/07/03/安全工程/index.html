安全工程

Posted on 2019-07-03 | In IE事务所 | | heat: ℃

第一章 绪论

1.安全的定义(P2)

是指在生产、生活系统中，能将人员伤亡或财产损失的概率和严重度控制在可接受水平之下的状态。

2.科学的安全的概念的含义(P2)

  • 安全是相对的

  • 安全是主观和客观的统一

  • 安全需要以定量分析为基础

3.危险与安全的关系(P3)

S=1-R

任何实际系统总是处于两者之间，包含一定的危险性和一定的安全性，可以用介于左右两圆中的一条垂线表示，垂线的上半段表示其安全性，下半段表示其危险性。当实际系统处于“可接受的安全水平”线(图中虚线)的右侧时，这样的系统是安全的。

4.风险的定量描述公式及方法(P5)

风险的定量描述

R=P X C

风险矩阵和风险矩阵图

5.事故的特性(P8)

  • 因果性
  • 随机性
  • 潜伏性

6.事故发生的概率与后果严重程度的关系(P10)

对于许多种类的事故，其发生频率与后果严重度之间有如下公式成立：

PCk=n

P——后果严重度达到C时事故的发生频率；

C——事故后果的严重度；

k——常数；

n——常数。

比例1：29：300称为海因里希法则

7.能量意外释放理论(P15)

吉布森、哈登等人提出事故是一种不正常的或不希望的能量释放并转移于人体

a)接触了超过机体组织(或结构)抵抗力的某种形式的过量的能量；

b)有机体与周围环境的正常能量交换受到了干扰(如窒息、淹溺等)。

8.两类危险源(P18)

系统中存在的、可能发生意外释放的能量或危险物质称为第一类危险源。

导致约束、限制能量措施失效或破坏的各种不安全因素称为第二类危险源。

9.第二类危险源的三个方面(P19)

包括人、物、环境三个方面。

(1)人的因素问题： “人失误”

(2)物的因素问题：物的不安全状态与物的故障

(3)环境因素问题

10.两类危险源之间的关系(P20)

在事故的发生、发展过程中，两类危险源相互依存、相辅相成。第一类危险源
在事故时释放出的能量，决定事故后果的严重程度;第二类危险源出现的难易程度决定事故发生的可能性的大小。两类危险源共同决定危险源的危险性。

11.工程的特征及负效应(P21)

工程的特征：目的性，技术性，实体性，群体性

工程的负效应：目的的偏离，技术的危害，实体的隐患，群体的威胁

12.安全工程的概念(P23)

安全工程是指在具有人类安全需要的领域,运用各种安全技术及其综合集成,以保障人体、设施、财产等动态安全的技术、方法、手段、措施等。

第二章 系统安全工程基础

1.系统生命周期的安全变化(P40)

H-系统的风险 t-时间

2.本质化安全工程(P41)

(1)本质化安全问题 （系统含有内在的能够从根本上防止发生事故的功能）

本质安全是系统安全的根本保证，从系统的构思、设计开始就融入系统，使系统含有内在的能够从根本上防止发生事故的功能。

(2)工程化安全问题（应用工程化的安全保护设备加强系统的安全性）

工程化安全思想是对本质安全的补充，其主导思想就是应用工程化的安全保护设备进一步加强系统在其生命周期中的安全性，同时必须确保工程安全设备在系统出现问题时不产生故障

3.PDCA循环模式(P43)

1）四个阶段

策划(planning)、实施(do)、监测(cheek)、评审(action)

2)八个步骤 是四个阶段中主要内容的具体化。
第一步，调查现状；
第二步，分析原因；
第三步，找出主要原因；
第四步，制订计划和活动措施；
第五步，即实施(D)阶段，按预定的计划认真执行；
第六步，即检查(C)阶段，调查了解采取对策后的实际效果；
第七步，根据检查的结果进行总结；
第八步，把本次循环没有解决的遗留问题，转入下一次PDCA循环中去。

4.危险源辨识的内容(P45)

危险源的类别，危险源的变化，可能产生的新危险源

5.危险性评价的内容 (P47)

事故发生概率的确定，事故后果严重度的确定 ，与安全标准的比较

6.危险控制的内容(P47)

预防事故发生的危险控制技术

1)根除危险因素
2)限制或减少危险因素
3)隔离、屏蔽和连锁
4)减少故障及失误
5)故障一安全措施
6)安全技术标准

防止或减少事故损失的危险控制技术

1)隔离
2)薄弱环节(接受小的损失)
3)个体防护
4)避难和援救

7.系统安全分析方法(P55)

(1)安全检查表法（SAFETY CHECKLIST）
(2)预先危险性分析（PHA）
(3)故障类型和影响分析（FMEA）
(4)危险性和可操作性研究（HAZOP）
(5)事件树分析（ETA）
(6)事故树分析 （FTA）

第三章 化工安全工程

1.化学能释放的危险性(P92)

化学能释放可能引起火灾和爆炸
化学能释放可能引起人体毒害
化学能释放可能引起装备腐蚀和损坏

2.燃烧的必要条件和充分条件(P93、94)

可燃物、助燃物和点火源是燃烧的三个基本要素，是发生燃烧的必要条件。

充分条件：必须使连锁反应不受抑制，燃烧时可燃物与助燃物要达到一定的比例，点火能量要足够

3.预防火灾事故的基本原则(P95)

消除着火源，控制可燃物，隔离空气，防止形成新的燃烧条件

4.均相燃烧和非均相燃烧(P94)

按照可燃物和助燃物相态（固、液、气）的异同，可分为均相燃烧和非均相燃烧。
均相燃烧是指可燃物和助燃物间的燃烧反应在同一相中进行。
非均相燃烧是指可燃物和助燃物并非同相。

5.爆炸的两大主要危害因素(P98、99)

压力波，抛射物

6.防火防爆设施(P131)

阻火装置
(1)安全液封
(2)水封井
(3)阻火器
(4)单向阀
(5)火星熄灭器
(6)阻火闸门

泄压装置（P135）
(1)安全阀
(2)防爆片
(3)防爆帽和易熔塞
(4)防爆窗
(5)排气管
(6)阻火闸门

7.灭火器材的类型和使用(P147)

泡沫灭火器使用时需要倒置稍加摇动，而后打开开关对着火焰喷出药剂;二氧
化碳灭火器只需一手持喇叭筒对着火源，一手打开开关即可;四氯化碳灭火器只需
打开开关液体即可喷出:而干粉灭火器只需提起圈环干粉即可喷出。

第四章 建筑安全工程

1.建筑施工能量释放的危险性-事故类型(P92)

1）势能释放的危险性：人员意外跌落–高空坠物事故，物体意外跌落–物体打击事故，边坡和脚手架势能释放–坍塌事故

2）动能释放的危险性

3）电能释放的危险性

2.高空作业要求(P164)

(1)要加强各级人员的安全教育和安全技术培训；

(2)安全防护用品配备到位，正确使用

(3)患有疾病，疲劳过度、精神不振的人员不适合高处作业，酒后不得从事高处作业

(4)六级以上强风天气，大雨、雪、雾天气，不得从事高处作业。

(5)安全网设置要求

1)凡4 m以上建筑施工工程，在建筑的首层要设一道3～6 m宽的安全网。

2)如果施工层采用立网作为防护，应保证立网高出建筑物1 m以上，且立网要搭接严密。

3)保证安全网规格质量和使用安全性能。

3.施工标志(PPT)

1.禁止标志：不准或制止人们不安全行动的图形标志。

2.警告标志：提醒人们对周围环境引起注意的图形标志。

3.指令标志：强制人们必须遵守某项规定。做出某种动作或采用防范措施的图形标志。

4.提示标志：向人们提供某种信息的图形标志

4.火灾逃生器材(PPT)

救生绳，呼吸器，灭火毯，手电筒，急救包，灭火器

5.火灾逃生方法(PPT)

  • 迅速判断火势的来源，朝与火势趋向相反的方向逃生。

  • 烟雾弥漫时，要用湿毛巾捂住口鼻。尽可能压低身体爬行并沿着墙壁边缘逃生，以免逃错方向。

  • 身上着火，就地打滚，扑压身上的火苗。如果近旁有水源，可用水浇或者跳入水中

  • 利用绳索，被单、衣物逃生，利用落水管、避雷针引下线逃生，利用天窗逃生

第五章 机械安全工程

1.主要的安全装置(P200)

1）固定安全装置，2）联锁安全装置，3）控制安全装置，4）自动安全装置，5）隔离安全装置，6）可调安全装置，7）自动调节安全装置，8）跳闸安全装置，9）双手控制安全装置

（固定联锁，自动控制，隔离可调，自动调节，双手控制跳闸）

2.机械伤害事故类型及对应部件(P201-202)

(1)卷绕和绞缠，作回转运动的物体(如轴类零件)

(2)卷入和碾压，相互配合的运动部件，如两个齿轮之间、齿轮与齿条、皮带与皮带轮、链条与链轮进入啮合部位的夹紧点等。

(3)挤压、剪切和冲撞，做往复直线运动的物体，如刨床、剪切、冲压机。

(4)飞出物打击

(5)切割和擦伤
切削刀具的锋刃，零件表面的毛刺，工件或废屑的锋利飞边，机械设备的尖棱、利角和锐边，粗糙的表面(如砂轮、毛坯)等

(6)碰撞和刮蹭
物体结构上的凸出、悬挂部分(如起重机的支腿、吊杆，机床的手柄等)，长、大加工件伸出机床的部分等，都可能产生碰撞和刮蹭危险。

3.加工车间安全常识(PPT)

(1)正确穿戴好个人防护用品
(2)操作前要对机械设备进行安全检查，而且要先空车运转，确认正常后，方可投入运行。
(3)机械设备在运行中也要按规定进行安全检查。特别是对紧固的物件，检查其是否由于振动而松动，以便重新紧固。
(4)严禁设备带故障运行。
(5)机械安全装置必须按规定正确使用，绝不能将其拆掉。
(6)机械设备使用的刀具、工夹具以及加工的零件等一定要装卡牢固，不得松动。
(7)机械设备在运转时，严禁用手调整，也不得用手测量零件，或进行润溅清扫杂物等。如必须进行，应首先关停机械设备。
(8)机械设备运转时，操作者不得离开工作岗位，以防发生问题时无人处理。
(9)工作结束后，应关闭开关，把刀具和工件从工件位置退出，并清理好工作场地，将零件、工夹具等摆放整齐，打扫好机械设备的卫生。

4.重新启动原则、定向失效模式(P209)

1）重新启动原则（将机械的自启动改为人工启动，防止动力中断接通后产生危险）

动力中断后重新接通时，如果机械设备自发启动会产生危险，就应采取措施，
使动力重新接通时机械设备不会自行启动，只有再次操作启动装置，机械设备才能运转

2）定向失效模式

这是指部件或系统主要失效模式是预先已知的，而且总是这些部件或系统失
效，这样可以事先针对其失效模式采取相应的预防措施。

5.压力容器的定义(P222)

压力容器是指盛装气体或者液体，承载一定压力的密闭设备并且满足以下三个条件：

  1. 工作压力大于或者等于0.1MPa（表压）

  2. 且压力与容积的乘积大于或者等于2.5MPa•L

  3. 液化气体和工作温度高于或者等于标准沸点

6.压力容器的失效模式(PPT)

韧性破裂，脆性破裂，疲劳破裂，腐蚀破裂，蠕变破裂

7.压力容器的常见附件(PPT)

安全阀，爆破片，压力表，液面计，温度计

第六章 电气安全工程

1.电气事故的类型(P252，PPT)

由设备或线路损坏造成停电导致的停产事故
人员触电造成的伤亡事故
由于电气原因造成的火灾

2.电击的三种类型(PPT)

  1. 单相点击：指人在地面或其它接地体上，人体的某一部位触及一相带电体时的触电 。
  2. 跨步电压电击：由于散流场内地面上的电位分布不均匀，人的两脚间电位差称为跨步电压 。
  3. 两相电击：是指人体两处同时触及两相带电体时的触电

3.安全电压(PPT)

4.电流对人体伤害的4个因素(PPT)

5.触电急救方法(PPT)

迅速设法切断电源；
用干燥的木棒、木板、绳索等不导电的东西设法解脱触电者；救护人员也可站在绝缘垫上或干木板上，绝缘自己进行救护；救护人最好用一只手进行救援。

判断是否假死，死亡的特征：
呼吸心跳停止；瞳孔放大；血管硬化；出现尸斑；尸僵

心肺复苏是心肺骤停的急救措施，主要包括：
畅通气道；口对口人工呼吸；胸外按压。

6.静电的预防(PPT)

①接地和屏蔽
②增湿和添加剂
③静电中和器
④工艺控制

7.雷电的破坏形式(P273)

1）直击雷

2）感应雷 ：静电感应雷，电磁感应雷，地电位反击

3）球雷

2）浪涌：电源浪涌，信号系统浪涌

8.建筑防雷和人身防雷(PPT)

建筑防雷

1）接闪器：避雷针，避雷线，避雷网，避雷带

2）引下线

3）接地装置

4）消雷器

人体防雷
①不要站在山顶，山脊等高处和躺在地上
②不要站在大树下，树林边或草垛旁躲雨
③不要靠近孤立的高楼、烟囟、电杆行走
④不要靠近金属物体
⑤不要站在避雷针附近
⑥不要使用移动电话
⑦可进入宽大金属构架或防雷设施的建筑物、汽车躲避

第七章 职业病危害

1.粉尘的预防和防护（P293）

密闭尘源；抽风除尘；湿式作业；个人防护；测定粉尘浓度和分散度；定期健康检查

消除毒物
密闭、隔离有害物质污染源
通风、湿法、滤除设备与措施
提高机体抵御能力
加强对有害物质的检测
定期健康检查
加强有害物质及预防措施的宣传教育

4.噪声的危害和防治（PPT）

噪声对人体健康危害主要表现在：

损伤听力，造成噪声性耳聋；
导致大脑皮层兴奋和平衡失调，脑血管功能损害，导致神经衰弱；
损伤心血管系统，引发消化系统失调，影响内分泌；
干扰人们正常的生活、休息、语言交谈和日常的工作学习，分散注意力，降低工作效率。

噪声的防治：降低声源，限制噪声传播，阻断噪声接收
