凸轮机构的压力角

压力角： 凸轮作用在从动件上的正压力与该力作用点B速度方向间所夹的锐角a

为了保证凸轮机构正常工作，要求： a < [a]

  • [ α ]= 30˚ —- 直动从动件
  • [ α ]= 45° —- 摆动从动件
  • [ α ]= 70°～80° —- 回程

习题

  • 凸轮机构主要是由凸轮，从动件和固定机架三个基本构件所组成。
  • 按凸轮的外形，凸轮机构主要分为盘形、移动凸轮等基本类型。
  • 从动杆与凸轮轮廓的接触形式有尖顶、滚子和平底、球面底四种。
  • 凸轮机构能使从动杆按照凸轮轮廓曲线，实现各种复杂的运动。
  • 以凸轮的理论轮廓曲线的最小半径所做的圆称为凸轮的基圆。
  • 凸轮理论轮廓曲线上的点的某点的法线方向(即从动杆的受力方向）与从动杆速度方向之间的夹角称为凸轮在该点的压力角
  • 凸轮机构中，凸轮基圆半径愈大，压力角愈小，机构传动性能愈好。

齿轮机构

渐开线齿廓

渐开线的形成和特性

  1. 渐开线的形成

    一条直线在圆上作纯滚动时，直线上任一点的轨迹 –渐开线

    BK – 发生线 基圆 – r{b} θ{k} – AK段的展角

  2. 渐开线的特性

    • 弧AB = BK;

    • B点为曲率中心，BK为曲率半径。渐开线上任意点的法线切于基圆，起始点A曲率半径为0；

    • 渐开线齿廓上某点的法线，与齿廓上该点速度方向线所夹锐角a{k} – 该点的压力角。 cosa{k} = OB/OK = r{b}/r{k}

      离中心越远r{k} ↑，渐开线上的压力角越大

    • 渐开线形状取决于基圆直径大小，当r{b} → ∞，变成直线

    • 基圆内无渐开线

渐开线齿廓的啮合特性

  1. 渐开线齿廓满足定传动比

两齿廓在任意点K啮合时，过K作两齿廓的法线N{1}N{2}，是基圆的切线，为定直线。
i {12} =ω{1}/ω{2}=O{2}C/ O{1}C = 常数

  1. 齿廓间正压力方向不变

    N{1}N{2}是啮合点的轨迹， 称为啮合线

    啮合线与节圆公切线之间所夹的锐角 α’ ，称为啮合角

  2. 运动可分性 △ O1N1C∽△O2N2C
    故传动比又可写成：
    i 12 =ω1/ω2= O2C/ O1C = rb2/rb1 －－基圆半径之反比。

齿轮各部分名称及渐开线标 准齿轮的基本尺寸

一、外齿轮

  1. 名称与符号
    齿顶圆－ da、ra
    齿根圆－ df、rf
    齿厚－ sk 任意圆上的弧长
    齿槽宽－ ek 弧长
    齿距 （周节）－ pk= sk+ek 同侧齿廓弧长
    法向齿距 （周节）－ pn = pb
    齿顶，齿顶高 ha
    齿根，齿根高 hf
    齿全高 h = ha + hf
    齿宽 – B
    分度圆－－具有标准模数和标准压力角的圆。(pk / π 为标准值）

  2. 基本参数

    • 齿数 – z

    • 模数 – m

      πdk = pk –> dk = pkz / π

      m = p / π

      分度圆 d = mz

    • 齿顶高系数 ha*

      顶隙系数 c*

    • 分度圆压力角

      由d=mz 知：m和z一定时，分度圆是一个大小唯一确定的圆。

      规定标准值：α＝20°
      由db＝dcosα可知，基圆也是一个大小唯一确定的圆。

一对渐开线齿轮的正确啮合条件是它们模数和压力角应分别相等。

习题

  • 渐开线上各点压力角等于a = arccos( rb / r ) ；越远离基圆，压力角越大, 基圆压力角等于 0
  • 如果分度圆上的压力角等于20°，模数取的是标准值，齿顶高系数和顶隙系数均为标准值，齿厚和齿间宽度相等的齿轮，就称为标准齿轮
  • 齿轮传动的标准安装是指节圆分度圆重合
  • 按标准中心距安装的渐开线直齿圆柱标准齿轮，节圆与分度圆重合，啮合角在数值上等于分度圆上的压力角
  • 渐开线标准直齿圆柱齿轮连续传动的条件是： 重合度大于等于 1
  • 齿轮传动的重合度越大，表示同时参与啮合的轮齿数目多 ，齿轮传动越平稳
  • 由于渐开线齿轮的传动比等于基圆半径的反比，所以即使两轮的安装中心距略有偏差，也不影响两轮的传动比
  • 相啮合的一对直齿圆柱齿轮的渐开线齿廓，其接触点的轨迹是一条直线
  • 标准外啮合斜齿轮传动的正确啮合条件是：两齿轮的法面模数和法面压力角都相等，齿轮的螺旋角角相等而旋向相反
  • 渐开线上各点的压力角不相等，基圆上压力角等于零
  • 对于齿数相同的齿轮，模数越大 ，齿轮的几何尺寸和齿轮的承载能力越大
  • 正常标准直齿圆柱齿轮的齿根高比齿顶高大
  • 一对齿轮啮合时,两齿轮的节圆始终相切
