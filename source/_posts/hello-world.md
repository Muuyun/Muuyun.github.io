# Hexo

Hexo的命令极简单，安装后只需要记住四个常用的即可。执行命令需要Git当前处于blog文件夹根目录下。

## 常用命令

### Create a new post

``` 
$ hexo new "My New Post"
```

### Run server(启动服务器)

``` 
$ hexo server
```

### Generate static files(生成静态文件)

```
$ hexo generate
```


### Deploy to remote sites(部署网站。部署网站前，需要预先生成静态文件)

```
$ hexo deploy
```

### clean 清除缓存文件 (db.json) 和已生成的静态文件 (public)
```
$ hexo clean
```
